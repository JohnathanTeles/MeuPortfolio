Projetos

Help Desk

O sistema tem como objetivo permitir que os usuários registrem chamados referentes a problemas técnicos, como falhas de hardware ou software. Cada chamado é visível apenas para o usuário que o criou, com exceção do administrador, que possui acesso a todos os chamados.

Time Clock

O LifeClock é uma ferramenta simples e interativa que permite calcular a idade de uma pessoa a partir da sua data de nascimento. Além de exibir a idade em anos, meses e dias, também fornece informações sobre o total de dias, horas, minutos e milissegundos vividos até o momento. A interface é intuitiva e fácil de usar, proporcionando uma experiência prática e eficiente aos usuários.

Game Shop

Trata-se de uma loja de jogos online que permite aos usuários explorar um catálogo de produtos, adicionar itens ao carrinho, modificar quantidades, remover produtos e concluir a compra de maneira intuitiva. O sistema também inclui funcionalidades de cadastro e autenticação de usuários.

Calculadora

Projeto de calculadora interativa com funcionalidades de operações matemáticas. Permite que os usuários realizem cálculos utilizando números e operadores aritméticos como soma, subtração, multiplicação e divisão. A interface da calculadora é simples e intuitiva, com a opção de limpar o campo de entrada a qualquer momento.
